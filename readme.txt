Inserts a meta box when edit pages for insert a Feed URL to show on front-end into a standard page or using a custom page template.

== Description ==

Version 1.0
First version of the plugin.

== Installation ==

1. Upload folder to the `/wp-content/plugins/` directory.
2. Activate the plugin via the Plugins menu.
3. select options on Wordpress editing page screen.
